# ⚠️ Archived Branch — Historical Reference Only

**This branch (`replit`) is a frozen snapshot of an early prototype and will not be maintained.**

## What this is

This is the original version of Fulfill, built on Replit using the base44 scaffolding. It predates the current monorepo, the Express API, Drizzle ORM, Supabase auth, and the local-first architecture.

## It will not run

- Dependencies are pinned to old versions and likely broken
- There is no `.env` / secrets configuration here
- The database schema and API do not match the current codebase

## Why it still exists

It is kept purely as a historical reference — a record of where the project started. The design decisions, early data model choices, and original feature scope may be of interest.

## Where to find the live code

The active codebase is on the `main` branch.
